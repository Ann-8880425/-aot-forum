# ⚔️ 进击的巨人 · 自由之翼论坛

**Attack on Titan Fan Forum** - 一个完整的进击的巨人主题动漫讨论论坛。

## 功能特色

- 🗳️ **角色投票** - 为你最喜爱的角色投票，实时排行
- 📖 **剧情分析** - 四季完整剧情简介与深度分析
- 🗺️ **故事线时间轴** - 2000年历史脉络完整梳理
- 💬 **论坛讨论** - 发帖、回复、点赞，分类讨论

## 本地运行

```bash
python server.py
```

然后访问 http://localhost:8090

## 技术栈

- 纯前端：HTML5 + CSS3 + JavaScript
- 数据持久化：localStorage
- 服务器：Python http.server
